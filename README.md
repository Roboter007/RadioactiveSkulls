<h1 style="text-align: center;"><span style="text-decoration: underline;">What is Radioactive Skulls?</span></h1>
<hr>
<p style="text-align: center;">This&nbsp;mod makes it possible to see skulls/heads through blocks. There are also some experimental options that could lead to a crash and are therefore disabled by default. These can be turned on in the config menu. <em><strong>As this can be considered cheating on some servers, I am not responsible for any bans related to this mod. Therefore, use it at your own risk in multiplayer!</strong></em></p>
<hr>
